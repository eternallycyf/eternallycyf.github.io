创建一个构造函数

原型与原型链

  • 原型与原型链指向关系
    • 可以很清晰的看到 实例对象test的__proto__直接指向 构造函数 Test.prototype
    • 同时 构造函数 Test.prototype.__proto__ 指向 Object.prototype
    • 而 Object.prototype.__proto__ 最终指向 null
    • 最终呈现了一个链状的结构
  • 构造函数与Function Object 内置对象的关系
    • Function 和 Object 既是函数又是对象
    • 由于Function.__proto__ === Function.prototype,这是一个内置的规则
    • 所以 Object的 __proto__ 指向了他的构造函数 即是 Function.prototype
    • Function 是所有构造函数的基类
  • constructor
    • 每个prototype 下都有一个 constructor
    • 而实例对象的 constructor 直接指向构造函数对象

new

  • 用于实例化构造函数
  • 一旦new了就是基本包装类型
  • 除了 new Object 指向引用是全等的
  • 其他都是不全等的,两个等号会自动转换

手写new

  • new 做了什么
    • 在内存中开辟一个新的空间,存储创建的新的对象
    • 把this设置为当前的对象
    • 设置对象的属性和方法的值
    • 把this这个对象返回
