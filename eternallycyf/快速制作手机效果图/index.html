快速制作手机效果图

发表于2020-04-12|更新于2020-04-12|tools
|字数总计:296|阅读时长:1分钟|阅读量:

网址

  • 打开这个网址 它提供了上千个外观样式 只需要上传张图片就可以自动合成了
  • https://smartmockups.com/zh/mockups
  • 具体使用方法如下

start

  • 随便打开一个链接 比如这个 https://smartmockups.com/zh/mockup/kXZo1wxohDQ
  • 点击从以下位置上传图片
  • 就可以发现图片合成成功了
  • 但是这里有两个问题需要解决
    • 1.高分辨率图片要会员
    • 2.图片有水印

高分辨率图片要会员

  • 首先上传图片让他合成后。再进行以下操作
  • 键盘点击 F12 进入开发者程序
  • 点击右上角的设置图标
  • 找到 Debugger 标题的哪一项
  • 勾选 Disable JavaScript
  • 然后直接可以在图片上右键进行本地存储了
  • 注意下载完后记得解除禁用 Disable JavaScript

2.图片有水印

  • 我查看他的相关网络请求
  • 他的流程是 用户上传图片后前端调用接口。后端返回有水印的合成图片
  • 因此紧靠浏览器不能解决掉
  • 目前只能使用免费版的了
  • 自己可以用ps把水印清除掉
