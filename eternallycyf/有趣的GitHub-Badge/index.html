有趣的GitHub-Badge

发表于2021-04-10|更新于2021-04-10|tools
|字数总计:255|阅读时长:1分钟|阅读量:

导读

  • 大家一定在浏览 github 的时候有看到过这样漂亮的徽标
  • 其实他的使用方式也非常简单

使用方式

  • Simple Icons官网图标列表
    • 这里可以看到各式各样的图标
    • 而他的名字就是后续配置需要的logo名
    • 此外基础配色也最好以它的为准 比较美观
  • 此外。还有些细节配置
    • 徽章实体风格: style= plastic | flat | flag-square | for-the-badge | social
    • 外观有不同的效果 自己可以挨个尝试下
  • 其次 badge 后边 也可以直接跟一个 svg 地址

github主页使用

  • 创建一个和自己github名字 同名的仓库
  • 在这个仓库里创建 README.md
  • 在这个里面的内容就自动展示在首页啦
