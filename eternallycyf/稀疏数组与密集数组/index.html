稀疏数组与密集数组

发表于2020-06-03|更新于2020-06-03|JavaScript
|字数总计:244|阅读时长:1分钟|阅读量:

稀疏数组

  • 数组的元素之间可以有空隙，因为一个数组其实就是一个键值映射
  • es5
    • forEach(), filter(), reduce(), every() 和some()都会跳过空位。
    • map()会跳过空位，但会保留这个值
    • join()和toString() 会将空位视为 undefined，而 undefined和 null 会被处理成空字符串。
  • es6
    • rray.from 将数组的空位转为 undefined
    • … 将空位转换为 undefined
    • copyWithin 将空位一起拷贝
    • fill() 将空位视为正常的数组位置
    • for…of 循环会遍历空位
    • entries()、keys()、values()、find()和findIndex() 将空位处理成 undefined

密集数组

  • Array.apply(null,Array(3));
    • Array 构造函数 当只有一个参数且为数字类型时 数组没有包含实际的元素
    • apply 将空值 转换成了数组
  • 密集数组可以遍历并赋值
